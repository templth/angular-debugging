Debugging Angular applications

This workshop describes how to leverage browser tools to identify, understand and fix errors.

List of exercices
1 - First bugs
This module describes how to identify and understand errors.
2 - Debugging Angular scopes
This module describes how to scopes (and scope hierarchy) work.
3 - Debugging dependency injection
This module describes how to debug dependency injection problems.
4 - Debugging HTTP
This module describes how to debug HTTP calls.
5 - Debugging forms and in-memory DOM
This module describes how to debug forms and interact with the in-memory DOM.
6 - Debugging directives
This module describes how to debug custom directives using ng-inspector and the JavaScript console.
